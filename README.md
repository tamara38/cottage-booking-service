# Cottage Booking System

Full-stack web application for reserving cottages with separate roles for **Users**, **Owners**, and **Admins**.

---

##  Features
- User registration & login (custom SQL authentication)  
- Role-based access (User, Owner, Admin)  
- Cottage listings with images  
- Cottage search & filtering  
- Booking system with availability check  
- Admin moderation panel (ban users, approve content, etc.)  
- Owners can manage cottages and reservations  
- Statistics dashboard for owners (weekend/monthly bookings)  
- Users can leave reviews with comments and ratings  
- Image upload for users and cottages  
- MySQL database

---

##  Tech Stack

**Backend**  
- Java (Spring Boot)  
- MySQL (classic SQL queries)  
- REST API  

**Frontend**  
- Angular (TypeScript, HTML, CSS, Bootstrap)  
- Communicates with backend via REST endpoints
